# 2025 MC_hackathon_Google

## Backends env setup

### venv
```
python3 -m venv venv
source venv/bin/activate
```

### install package
```
pip install -r requirements.txt
```

### setting API key and activate backend
```
export GOOGLE_API_KEY="YOUR_API_KEY"
python -m app.main
```
